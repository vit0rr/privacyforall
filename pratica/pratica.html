Como começar?

Aqui é o começo, onde você irá com a prática defender sua privacidade da forma correta. Para que isso ocorra de forma saudável, será dividido em técnicas iniciais/intermediárias para usuários comuns, e avançada aos usuários avançados.



DNS


Domain Name Server

DNS é o resposável por permitir uma memorização mais fácil dos sites que entramos. Para exemplificação, é o que "traduz" um IP(123.456.789) em um endereço(www.google.com). Mas o que isso tem com privacidade?

Caso seu DNS não esteja configurado da forma correta, com certeza há o vazamento de DNS ocorrendo. Que consiste numa falha onde permite as solicitações de DNS sejam reveladas aos servidores de DNS do ISP. Ou seja, a sua provedora de internet sabe exatamente quais são suas atividades online.

Se tiver interesse, você pode realizar os testes de DNS Leak. Caso os testes retorne seu Hostname e ISP da provedora, provavelmente está tendo vazamento de DNS.

Como contornar essa falha?

É simples, você pode obter um novo endereço de DNS com AdGuard, UncensoredDNS ou LibreDNS. O processo é o mesmo em todos, então vou dar o exemplo com o AdGuard.

Depois de clicar no link, clique no botão verde "Como configurar?" e você será redirecionado para uma página como essa:



Aqui você pode escolher, configurar somente para o seu computador, celular ou roteador. A diferença é que, caso configurado para o seu computador, por exemplo, apenas ele terá sua conexão segura, o que não garante a privacidade de todos os outros dispositivos da rede. Então é recomendado seguir os passos de "Roteador". As instruções do próprio site são claras, e caso tenha dúvidas específicas pode pesquisar pelo seu modelo de roteador na internet para saber como trocar o DNS dele.
